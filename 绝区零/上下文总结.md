# 绝区零 RPG 角色卡 · 上下文总结

## 项目概述
在 `d:\Lore\CharCard\绝区零\` 下创建一套面向 SillyTavern 的《绝区零》角色卡，支持**法厄同**和**非法厄同**两种开局模式，搭配完整的MVU变量系统和可视化面板。

已推送至 `origin/main`（commit `9c506cf`），但**信息面板.html 的最新修改尚未构建和推送**。

## 已完成的构建流程

1. `build-all.js` 启动时：
   - 读取 `MVU/` 下7个面板JS → 生成初始变量YAML + 更新规则 + 面板触发词条目
   - 生成世界书YAML文件到 `cards/绝区零/世界书/`
   - 构建102条 world entries
   - 读取 `面板/` 下HTML文件 → 生成 `regex_scripts`（2个：角色创建+信息面板）
   - 读取 `开场白/` 下7个txt → alternate_greetings
   - 输出 `cards/绝区零/绝区零.json`

2. 角色卡JSON结构：
   - `spec: chara_card_v3`
   - `data.extensions.regex_scripts` 包含2个条目：
     - `<character_creation>` → 角色创建面板HTML
     - `<info_panel>` → 信息面板HTML
   - `data.character_book.entries` 102条
   - 第102条为法厄同开关（role=2）

## 使用说明

- 运行 `node build-all.js` 构建角色卡
- 在 SillyTavern 中导入 `绝区零/cards/绝区零/绝区零.json`
- 法厄同开局：选择 alternate greetings 0-3
- 非法厄同开局：选择 alternate greetings 4-6（自动弹出角色创建面板）
- 对话中输入 `<info_panel></info_panel>` 可查看实时状态面板
