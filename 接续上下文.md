## 一、项目概述

为 SillyTavern 创建《绝区零》RPG 角色卡（支持法厄同/非法厄同双开局 + 完整 MVU 变量系统 + 可视化面板）。

**构建命令**：
```bash
cd 绝区零
FORGE="$HOME/.zcode/skills/tavern-cards/scripts/tavern-cards-forge.mjs"
node "$FORGE" pack 绝区零
cd cards/绝区零 && node post-pack.js   # 后处理（必跑）
```

**输出**：`绝区零/cards/绝区零/绝区零.json`（140 条目 + 6 正则 + 2 脚本）

## 三、关键规则（接续任务时注意）

### 修改流程（纯 forge，不跑 build-all.js）
```
改 yaml → forge pack → post-pack.js
```

### 角色条目格式（角色扮演向，不含游戏数值）
```yaml
name/role/gender/height/type/appearance/personality/weapon/background/info/sp_form
```
禁止：attribute/specialty/rank/倍率/驱动盘/音擎/影画/配队

### 着装禁止拼接
6 格着装每个槽只写一个物品名，禁止 "A+B" 写法

### 已修复的坑
- 哲不戴眼镜（wiki 无此设定）
- 铃的穿着：黑色连帽无袖马甲+灰色百褶裙+不对称腿饰+R字发卡
- 猫又鞋子→彩色护踝带（非"赤脚+护踝带"）
- 牲鬼→非牺牲
