# 夜鹄 · SillyTavern 角色卡

## 项目定位
都市灵异·志怪 RPG，玩家扮演「已故灵管局局长在阳间当书店老板」的 沈夜。

## 项目结构
```
夜鹄/
├── .cardrc.json                 # forge 配置
└── cards/夜鸮/
    ├── schema.ts                # Zod 变量结构
    ├── tavern-cards-state.json  # forge 状态
    ├── 夜鸮.json                # ← 输出卡
    ├── post-pack.js             # 后处理脚本
    ├── 正则/
    │   ├── 变量更新美化.html    # 角色创建/初始化面板
    │   └── 状态栏界面.html      # 信息面板
    ├── 开场白/
    │   └── 0.txt
    ├── 脚本/
    │   ├── MVU.txt
    │   └── Zod.txt
    └── 世界书/
        ├── 变量/
        │   ├── initvar.yaml
        │   ├── 变量更新规则.yaml
        │   └── 变量输出格式.txt
        ├── MVU/                 # 变量系统双重副本
        │   ├── 初始变量.yaml
        │   ├── 变量更新规则.yaml
        │   └── 变量输出格式.yaml
        ├── EJS/
        │   └── EJS预处理.txt
        ├── 世界观/ 地理/ 势力/
        ├── 角色/ NPC/
        ├── 扮演准则/
        └── 时间线/
```

## 打包流程
```
cd 夜鹄
node C:\...\tavern-cards-forge.mjs pack 夜鸮
cd cards\夜鸮 && node post-pack.js
```

## 设计原则
- 变量系统同步：改 schema → 同时更新 变量/ + MVU/ 两副本
- 面板只改 正则/ 目录
- 禁止 build-all.js（已废弃）
